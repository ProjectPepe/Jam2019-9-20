基本的にいじれそうな値はすべて
[WallTileMap]のところにPublicとして置いてあります。

[歩く速度]を変える場合：Walk Speed値　を変更してください。

[本棚が倒れたとき、キャラがびっくりして止まる時間]を変えたい場合 : Stop Time値　を変更してください。

Current Node Id は　キャラクターの現在地、つまり初期値に該当します。
その座標の値が知りたい時、
いったんプログラムを実行して、その場にマウスを当て、左クリックをすることでデバッグ値として左下に表示されます（x,y)
その値を初期値として入力。

Goal Node Id は　キャラクターが逃げる場所の値です。
プレイヤー側の勝利座標（死への道）はまだ実装できていません。
その座標を知りたいときもまた上と同じです。

マップを作るとき、

壁判定として得たいときは[Wall Tile Map]で [Tile Map]を色付けしてください。

[Floor Tile Map] はその名の通り歩けるマップチップを設定してください。
